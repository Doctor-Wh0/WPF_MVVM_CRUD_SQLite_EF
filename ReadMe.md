Необходимо установить проект DAL в качестве запускаемого проекта. Нажать на проект -> свойства -> Назначить ...
Далее
В меню Visual (там где Вид, Правка)
Средства -> Диспетчер пакетов NuGet -> Консоль диспетчера пакетов.
Внизу, вместо окна Дебага с выводом вылезет консоль диспетчера пакетов.
В меню этого окна найти туглбар "Проект по умолчанию" и назначить DAL.

Дальше вбить несколько комманд. 
Так как папка с миграциями уже есть, то достаточно команды: Update-Database
Если надо что-то поменять с моделями, то удаляем папку с миграциями и вводим команды:
Add-Migration InitialCreate
Update-Database
